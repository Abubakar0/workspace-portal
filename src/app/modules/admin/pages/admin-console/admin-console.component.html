Admin Powers

  • Create and edit employees with role, shift, team, and marketplace scope.
  • Provision VPS records and map them to Guacamole connection IDs.
  • Maintain team ownership for Amazon, eBay, development, and other operations.
  • Review audit logs for admin changes and workspace launches.

Employee Limits

  • Employees do not see the admin console.
  • Employees only receive workspace data scoped by backend permissions.
  • Employees cannot create users, edit VPS inventory, or view audit logs.
  • Managers can be modeled separately from admins for team-level access.
